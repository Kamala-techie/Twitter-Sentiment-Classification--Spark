#Implementation of Twitter Sentimental Analysis

Training data  - data/train.csv

Test data - data/test.csv

Implementation - lkotha2/Hw4_final.ipynb

Implemented in Python using Spark Map Reduce.

  Data Preprocessing - Regex cleaning, Stop word Removal , Porters stemming

  Feature Extractions - TFIDF vectors of unigrams, bigrams

  Machine Learning Models

      Algorithms used : 
      1. Logistic Regression
      2. Naive Bayesian Classifier
      3. Decision Tree

  10 Cross Validation

  Evaluation Metrics ( Precision, Recall, Fscore, ROC curve)
